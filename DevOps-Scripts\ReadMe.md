# UiPath & Alteryx DevOps Deployment Scripts - README

This repository contains a suite of **DevOps automation scripts** for building, testing, packaging, and deploying **UiPath RPA workflows** and **Alteryx analytics workflows** to their respective platforms:

- **UiPath Orchestrator** (cloud or on-prem)
- **Alteryx Gallery** (Server)

These scripts enable **CI/CD pipelines**, **version control integration**, and **repeatable deployments** across development, test, and production environments.
